# Coditional problem

### this Cpp program Given a positive integer , do the following

##### If 1<= n >=9 , print the lowercase English word corresponding to the number (e.g., one for 1, two for 2, etc.)

##### If n>9 , print Greater than 9

# Sample Input 0

    5

# Sample Output 0

    five

# Sample Input 1

    8

# Sample Output 1

    eight

# Sample Input 2

    44

# Sample Output 2

    Greater than 9
